# 🔐 SecLinks — Guia de Referência para Analistas de Segurança

Guia de consulta rápida com links curados para analistas de segurança, organizado por categorias e com busca em tempo real.

## ✨ Funcionalidades

- 🔍 **Busca em tempo real** — filtra por título, URL ou descrição
- 🏷️ **Filtros por categoria** — Blue Team, Red Team, Hunting, etc.
- 🌙 **Tema dark** — ideal para trabalho em SOC
- 📱 **Responsivo** — funciona em mobile e desktop
- ⚡ **Zero dependências** — HTML puro, sem backend

## 📁 Categorias incluídas

| Categoria | Descrição |
|-----------|-----------|
| 📚 Wiki & Guias | Repositórios e guias gerais de cybersecurity |
| 🎯 Threat Hunting | Ferramentas e técnicas de caça a ameaças |
| 🛡️ Blue Team | Defesa, monitoramento e resposta a incidentes |
| ⚔️ Red Team / Offensive | Ferramentas e técnicas ofensivas |
| ⬆️ Privilege Escalation | Técnicas de escalação Linux e Windows |
| 🪟 Windows & Active Directory | Logs, eventos, ACL e AD |
| 📡 ETW & Sysmon | Event Tracing for Windows e Sysmon |
| 💉 DLL & Injeção | DLL Hijacking e injeção de código |
| 🐧 Linux | Referências e ferramentas Linux |
| 🌐 Redes & Portas | Portas, DNS, WHOIS e redes |
| 🦠 Malware & Análise | Análise de malware e sandboxes |
| 🎣 Phishing & Engenharia Social | Frameworks de phishing |
| 🔎 Dorks & OSINT | Google Dorks e reconhecimento |
| 🏋️ HTB & Labs | HackTheBox e laboratórios práticos |

## ⚠️ Aviso

Este guia é destinado exclusivamente para fins educacionais e de uso profissional por analistas de segurança. Use com responsabilidade e dentro dos limites legais.
